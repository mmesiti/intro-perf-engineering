    "## Maximum performance and bandwidth estimates: the Roofline model\n",
    "\n",
    "The [Roofline Model](https://en.wikipedia.org/wiki/Roofline_model)\n",
    "is a performance model that assumes\n",
    "that there are 2 main performance bottlenecks: \n",
    "\n",
    "- peak floating point performance \n",
    "- the memory bandwidth.\n",
    "\n",
    "In this model, \n",
    "the relevant bottleneck for the kernel (typically, a loop) being investigated \n",
    "is determined by its **arithmetic intensity** (also **operational intensity**), \n",
    "that is the ratio between how many FP operations are done \n",
    "and the *memory traffic* in byte, \n",
    "that is how many bytes have to be moved \n",
    "from/to a particular memory system (cache level or RAM).\n",
    "\n",
    "![Roofline Model (Wikipedia)](https://upload.wikimedia.org/wikipedia/commons/thumb/b/b1/Example_of_a_Roofline_model.svg/1920px-Example_of_a_Roofline_model.svg.png) By <a href=\"//commons.wikimedia.org/w/index.php?title=User:Giu.natale&amp;action=edit&amp;redlink=1\" class=\"new\" title=\"User:Giu.natale (page does not exist)\">Giu.natale</a> - <span class=\"int-own-work\" lang=\"en\">Own work</span>, <a href=\"https://creativecommons.org/licenses/by-sa/4.0\" title=\"Creative Commons Attribution-Share Alike 4.0\">CC BY-SA 4.0</a>, <a href=\"https://commons.wikimedia.org/w/index.php?curid=49641314\">Link</a>\n",
    "\n",
    "The maximum possible FP performance for a kernel is given by that \"roofline\" \n",
    "(called so probably because it reminds of a roof):\n",
    "\n",
    "- kernels which require few operations per byte\n",
    "  (low arithmetic intensity, on the left side of the plot above)\n",
    "  will be ultimately **memory bound**\n",
    "- kernels which perform enough operations per byte\n",
    "  (right end of the plot above)\n",
    "  will be **compute bound**, that is able to run at the maximum FP performance of the machine.\n",
    "\n",
    "Most codes in use today are typically **memory bound**.\n",
    "\n",
    "Notice that if our code is *latency limited*\n",
    "its performance will stay way below the roofline.\n",
    "\n",
